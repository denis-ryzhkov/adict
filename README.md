adict
=====

Dict with attr access to keys.

Usage:

    pip install adict
    from adict import adict

    d = adict(a=1)
    assert d.a == d['a'] == 1

See all features, including `ajson()` in [`adict.py:test()`](https://github.com/denis-ryzhkov/adict/blob/master/adict.py#L46).
